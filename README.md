# Pulse

A real-time LiveChat application for seamless communication and instant messaging.

## 📋 Description

Pulse is a modern, responsive web-based chat application built with JavaScript, enabling users to connect and communicate in real-time. The application provides a lightweight and intuitive interface for instant messaging and live conversations.

**Live Demo:** https://chat.p196hql.qzz.io

## 🛠️ Tech Stack

- **Frontend:** JavaScript (97.9%)
- **Styling:** CSS (1.9%)
- **Markup:** HTML (0.2%)

## 📦 Setup & Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/p196hql/pulse.git
   cd pulse
   ```

2. **Install dependencies**
   ```bash
   cd web
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:3000` (or the port displayed in your terminal)

### Build for Production

```bash
npm run build
```
